# Self-Rescue —— 自我发掘计划

> 通过持续对话，系统性地发现自己的核心能力与优势，找到变现和职业发展的方向。

## 项目结构

```
self-rescue/
├── profile/                # 个人画像（持续迭代）
│   ├── skills.md           # 技能清单：硬技能 + 软技能
│   ├── experiences.md      # 关键经历与成就
│   ├── values.md           # 价值观与驱动力
│   └── strengths.md        # 核心优势总结（阶段性更新）
├── exploration/            # 探索记录
│   └── sessions/           # 每次对话的关键发现
├── insights/               # 洞察与分析
│   ├── skill-map.md        # 技能关联图谱
│   └── opportunities.md    # 潜在变现方向与机会
└── README.md
```

## 怎么开始？

**输入 `/start`，然后随便聊就行。**

不需要准备任何东西，不需要知道自己擅长什么。你只管说，AI 会从对话中发现你的能力线索。

## 可用指令（Skills）

### 第一层：自然对话（零门槛）

| 指令 | 用途 |
|------|------|
| `/start` | **从这里开始。** 随便聊，AI 自动捕捉技能线索 |

### 第二层：定向探索（有方向后）

| 指令 | 用途 | 示例 |
|------|------|------|
| `/explore` | 针对某个方向深入挖掘 | `/explore 技术能力` |
| `/reflect` | 从一段具体经历中提炼能力 | `/reflect 上次做的项目` |

### 第三层：分析输出（积累够了）

| 指令 | 用途 |
|------|------|
| `/assess` | 综合评估核心优势和独特组合 |
| `/opportunity` | 将能力映射到变现/职业方向 |

### 辅助工具

| 指令 | 用途 |
|------|------|
| `/status` | 查看探索进度，看看聊到哪了 |
| `/session` | 保存本次对话的发现 |

## 工作方式

```
/start       随便聊 → 发现线索
                ↓
/explore     深入某个方向 ← /reflect 从经历中提炼
                ↓
/assess      综合分析核心优势
                ↓
/opportunity 探索变现方向
                
/status 随时查看进度  |  /session 随时保存发现
```

## 阶段目标

- **Phase 1：广泛探索** — 尽可能多地发掘技能和经历（当前阶段）
- **Phase 2：深度挖掘** — 找到反复出现的模式和独特优势
- **Phase 3：方向聚焦** — 确定 2-3 个最有潜力的方向
- **Phase 4：行动计划** — 为选定方向制定具体的变现/发展路径
